## 1. Linha do Tempo

Representação visual dos principais marcos históricos das redes de computadores, desde os mainframes dos anos 1950 até as redes 5G e Wi-Fi 6 da década de 2020.

| Marco | Ano | Descrição |
|-------|-----|-----------|
| Mainframes | 1950s | Computadores centrais com terminais burros, sem redes |
| ARPANET | 1969 | Primeira rede com 4 nós: UCLA, SRI, UCSB e Utah |
| TCP/IP | 1983 | Protocolo padrão adotado, base da internet atual |
| World Wide Web | 1991 | Tim Berners-Lee cria HTTP, HTML e URLs |
| Internet Comercial | 1999 | Banda larga, e-mail corporativo, boom digital |
| Redes Móveis 3G | 2007 | iPhone lançado, internet no celular |
| Wi-Fi 6 & 5G | 2020s | Ultra-baixa latência, IoT, computação em nuvem |

## 2. Elementos da Rede

Esquema ilustrando os quatro elementos fundamentais de uma rede, com exemplo prático de compartilhamento de arquivo em rede Windows (protocolo SMB/TCP-IP).

| Elemento | Definição | Exemplo Real (Windows) |
|----------|-----------|------------------------|
| **Emissor** | Origem da mensagem/dados | PC com Windows 11 (pasta compartilhada) |
| **Meio de Transmissão** | Canal físico ou sem fio | Rede Wi-Fi 5 GHz (802.11ax) |
| **Protocolo** | Regras que governam a comunicação | SMB / TCP/IP (File Sharing Windows) |
| **Receptor** | Destino da mensagem/dados | Notebook Windows (acessa o arquivo) |

## 3. Classificação de Redes

Quadro comparativo dos quatro tipos de rede classificados por abrangência geográfica, com exemplos práticos no ambiente Windows.

| Tipo | Nome Completo | Abrangência | Velocidade | Exemplo Windows |
|------|---------------|-------------|------------|-----------------|
| **PAN** | Personal Area Network | Até 10 metros | Até 3 Mbps | Bluetooth Settings |
| **LAN** | Local Area Network | Prédio / campus | 10 Mbps – 10 Gbps | Rede e Internet > Wi-Fi |
| **MAN** | Metropolitan Area Network | Cidade / região | 1 Mbps – 1 Gbps | VPN corporativa / DirectAccess |
| **WAN** | Wide Area Network | Países / mundial | 56 Kbps – 100 Gbps | Windows Update / Azure Cloud |
